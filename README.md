# Sales-Performance-KPI
This project analyzes sales data to uncover insights on revenue, profit, customer segments, product performance, and quarterly trends. Using Excel, I transformed raw data into a simple sales performance dashboard to support business decision-making.

## Tools Used
- Microsoft Excel

## Key Focus Areas
- Revenue and profit analysis  
- Customer segmentation  
- Product performance  
- Quarterly sales trends  

## Project Overview
In this project, I worked with raw sales data and transformed it into a simple sales performance dashboard. The goal was to turn scattered numbers into meaningful business insights and strengthen my understanding of data analytics fundamentals.

## What I Learned
- Data cleaning and organization  
- Calculated fields (revenue and profit)  
- Basic dashboard design  
- Analytical thinking and problem-solving  

This project is part of my learning journey in data analytics and data science.
